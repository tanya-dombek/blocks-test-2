## Run Locally

Clone the project

```bash
  git clone https://github.com/tanya-dombek/blocks-test-2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

The server should now be running. You can access the application by opening a web browser and entering the following URL:

```bash
  http://localhost:5173/
```
